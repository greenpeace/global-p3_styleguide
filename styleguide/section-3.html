3.3 Labels

Label and input are wrapped in div. If required class is added, additional span with a @red-1 star is included.

.show-default

Default mode of showing the labels.

.horizontal-ra

Show labels in left side with right text alignment.

.horizontal-la

Show labels in left side with left text alignment.

.inline

Show label and input inline.
