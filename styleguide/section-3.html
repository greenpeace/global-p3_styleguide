3.1 Generic Form Item

Labels and input are wrapped in a div with the type of the class.

Compatible in FF2+, Chrome1+, IE?, Opera?, Safari?

Since version 3.0.

:hover

subtle hover highlight

:focus

green border

:disabled

grey out to indicate it cannot be used

<form action="#" id="action-form-pledgeName" method="post" accept-charset="utf-8" class="js-track-abandonment">

3.5 Textarea required and larger

Labels and input are wrapped in a div with the type of the class.

Compatible in FF2+, Chrome1+.

Since version 3.0

.required

Label and input are wrapped in div.text.required. Additional span with a star is included. Input also get the required class.

.large

Make the box bigger.

<div class="textarea {$modifiers}">
  <label for="ActionMessage">Your message</label>
  <textarea rows="5" id="ActionMessage" name="ActionMessage">
